# Security Policy

## Supported Versions

StudentCareer+ is currently a **concept and prototype project**.
There are no released or deployed versions at this stage.

| Version   | Supported |
| --------- | --------- |
| Prototype | Yes       |
| MVP       | No        |
| Production| No        |

---

## Reporting a Vulnerability

If you notice any potential security or privacy concerns related to this repository or the product concept, please report them responsibly.

You can report an issue by:
- Opening a GitHub Issue with a clear description, or
- Contacting the repository owner directly.

Since this project is in an early stage, reports will be reviewed on a best-effort basis.
This document describes a preferred communication process and does not represent a legal or operational commitment.
